# Student Grading System

A simple Python program that calculates student grades and provides feedback based on their marks.

## Features
- **Functional Logic:** Uses a dedicated function to process grades.
- **Dynamic Messaging:** Provides personalized encouragement based on performance.
- **Specific Grading Scale:**
  - A: 90-100
  - B: 80-89
  - C: 70-79
  - D: 60-69
  - F: Below 60

## How to Run
1. Ensure you have Python installed.
2. Run the script using:
   ```bash
   python grading_system.py
